After pushing the button '0'.

Press the button
Postcondition
Displey a number

After pressing the button '0' this function calls auxiliary function check_input_zero, after that it checks the lenght of the string (which is limited to 10 characters); if this condition is fulfilled it will put 0 to the string and the calculation process is displied on lcd display and label
